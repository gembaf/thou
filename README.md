## Installation

```sh
$ git clone https://github.com/gembaf/thou.git
$ cd thou
$ bundle install -j4 --path vendor/bundler
$ npm install
$ npm run build
$ bundle exec rake assets:precompile
$ bundle exec foreman start
```
